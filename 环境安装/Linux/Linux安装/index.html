使用 VMware 安装 Centos 7，并使用 Xshell 连接。

1. 下载Linux虚拟机镜像

下载地址：

本文下载版本为：CentOS-7-x86_64-DVD-2003.iso

2. 使用 VMware 安装

  1. 打开 VMware 新建虚拟机。

  1. 点击典型，再点击下一步。

  1. 稍后安装操作系统，再点击下一步。

  1. 选择操作系统和版本，再点击下一步。

  1. 选择虚拟机名称和虚拟机安装位置，再点击下一步。

  1. 指定虚拟机磁盘大小，如果安装时分配的内存不够，后期还可以继续改，再点击下一步。

  1. 选择自定义硬件。

  1. 根据自己电脑的配置选择，点击关闭之后会自动保存。

  1. 完成安装。

3. 启动虚拟机

  1. 在 VMware 中启动虚拟机。
  1. 进入虚拟机界面，点击鼠标，让鼠标进入虚拟机界面，然后使用 ↑ ↓ 箭头选择 Install CentOS 7。

  1. 选择简体中文，下一步。

  1. 安装配置，软件选择可以选择带图形界面的（像 Windows 一样的图形界面），最小安装只有命令窗口；安装位置选择自动分区，打开以太网连接。然后点击开始安装进行安装。

  1. 设置 root 密码和创建用户，如果密码级别较弱，需要连续点击两次完成按钮完成密码设置操作。

  1. 安装完成之后，点击重启。

  1. 使用 root 账户或者安装时创建的用户账号登录系统，此处密码默认不显示，需要自己确定输入正确。

4. 虚拟机设置

4.1 固定 IP 地址

5. Linux 出现的网络问题

CentOS网络服务起不来 Job for network.service failed
service network restart 和 systemctl start network.service 启动centos 网络服务出错，

原因：centos 系统上，目前有 NetworkManager 和 network 两种网络管理工具。如果两种都配置会引起冲突，而且 NetworkManager 在网络断开的时候，会清理路由，如果一些自定义的路由，没有加入 到NetworkManager 的配置文件中，路由就被清理掉，网络连接后需要自定义添加上去。
